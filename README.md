# Dev Nearby

Dev Nearby is an app to find the nearest developer

This repository has the entire app for you run.

# Stack

- Backend
  - NodeJs
  - Socket
  - Express
  - Mongoose

- Frontend
  - React
  - Axios
  - Socket.io

- Mobile
  - React Native
  - Expo
  - React Navigation
  - React Native Maps
  - Socket.io
